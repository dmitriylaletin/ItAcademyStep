1.	Реализовать конструктор Connection. Если IP-адрес или порт пустые, то выбрасывать исключение.

2.	Network должен иметь количество допустимых девайсов. Если устанавливается кол-во меньше одного,
то выбрасывать исключение. При добавление нового девайса в сеть реализовать проверку на допустимость его добавления,
если свободных мест нет - выбросить исключение.

3.	Для девайса реализовать переопределить метод toString, который будет показывать IP-address/Port девайса и имя сети,
 в которой он расположен

4.	В NetworkLAN реализовать методы pushData/getData, так, чтобы данные в виде строки переходили от одного объекта
 другому и возвращалось состояние отправки (успешно/провал)

5.	В Device реализовать методы request, который будет отправлять данные на другой девайс в сети,
 а response будет получать от него ответ.
