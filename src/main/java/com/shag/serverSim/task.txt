1. Используя интерфейс ServerSocket, реализовать сервер, который будет создавать клиентский сокет и выводить
в консоль информацию полученную от клиента из стандартного потока ввода

2. Используя интерфейс Socket, реализовать сервер клиента, который будет подключаться к серверу и
 передавать введенные из консоли данные через стандартный поток вывода

3. Если пользователь вводит определенный символ (выбираете самостоятельно), то сервер предлагает завершить сеанс.
Пользователь принимает и сервера перестаю работать, либо отклоняет и сеанс продолжается
